This package is supplied to help implementing the Calypso Audit Application.
It contains the following files:
- LowLevelAPI.h:   Calypso Portable Object Reader Interface for Example Audit Application
                   Interface to communicate to a Calypso card through a contactless reader.

- AuditApplication.c:  Example implementation of a terminal application for the Calypso
                       Registration Audit, as described in the document 090602-LT-AuditApplication

The AuditApplication.c may be used to implement the Calypso Audit Application.
The functions described in LowLevelAPI.h must be implemented for your specific reader.
